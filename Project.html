Portfolio

Apraksts:

Projekts tika izveidots kā pirmais projekts manā portfoliā. Tas palīdzēja man apgūt pamatus tīmekļa izstrādē un izveidot strukturētu darba procesu. Šī pieredze man deva iespēju praktiski pielietot HTML, CSS un JavaScript prasmes. Lai uzlabotu lietotāja pieredzi, projektā tika iekļautas responsīvas dizaina funkcijas. Projekts tika arī optimizēts, lai nodrošinātu ātru ielādes laiku un saderību ar dažādām pārlūkprogrammām. Šis projekts kļuva par pamatu tālākajai izaugsmei un sarežģītākiem projektiem manā portfoliā.
